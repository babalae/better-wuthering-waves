# IcoMoon CLI

> create.cmd

**Requests**

```bash
npm i -g icomoon-cli
```

Add svg icon to [icomoon app project](https://icomoon.io/app).

**Executes**

```bash
icomoon-cli -i 1.svg -s selection.json -n 1 -o output
```
